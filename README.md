## Projekt 1: Notizen Verwaltung -> Gruppe 9 ##

Wir haben eine mini Rest Applikation gemacht

Um es Lokal auszufüren:

    npm install -g grunt-cli
    
    npm install


Danach muss folgendes ausgefürt werden:

    grunt handlebars

Dann als letztes ist genügend das index.js im root verzeichniss zu starten als NPM server.
